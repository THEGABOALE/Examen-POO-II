## 2. Tema de la Aplicación
**MisNotasApp** es una aplicación Android nativa orientada a la gestión de notas personales. Su propósito es ofrecer una herramienta sencilla, rápida y visualmente clara para registrar ideas, recordatorios o listas de tareas cotidianas.

El proyecto está estructurado como una aplicación CRUD completa bajo el patrón arquitectónico MVVM y los lineamientos de Material 3, permitiendo crear, consultar, editar y eliminar registros desde una interfaz unificada y reactiva.

---

## 3. Funcionalidades Implementadas
* **Operaciones CRUD:** Creación, lectura, edición y eliminación completa de notas.
* **Control Temporal:** Asignación y actualización automática de la fecha actual al guardar cambios.
* **Mecanismos de Confirmación:** Diálogos modales para prevenir eliminaciones accidentales.
* **Validación de Campos:** Restricción de registros vacíos y asignación del valor 'Sin título' por defecto.
* **UI Reactiva:** Estado alternativo para pantallas vacías y actualización automática de la interfaz ante cambios en los datos.
